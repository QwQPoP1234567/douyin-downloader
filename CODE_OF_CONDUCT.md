# 社区行为准则

参与本项目即表示同意以尊重、专业和建设性的方式交流。

## 期望行为

- 尊重不同经验、背景和观点；
- 针对代码和事实讨论，不攻击个人；
- 提供可复现的信息并保护他人隐私；
- 接受维护者为项目安全、合法使用和社区秩序作出的合理决定。

## 不可接受行为

- 骚扰、侮辱、歧视或威胁；
- 发布他人的个人信息、Cookie、Token、账号或下载内容；
- 鼓励绕过验证码、权限控制、付费限制或平台安全机制；
- 利用项目从事侵权、滥用或违反适用法律的活动。

## 执行

普通问题可通过 Issue 或 Discussion 反馈。涉及安全或隐私的内容请按照 [安全政策](SECURITY.md) 私密报告。维护者可以编辑、隐藏或删除违反本准则的内容，并限制持续违规者参与项目。
